Description détaillée

Creates a new "Terminal User Interface" using ncurses and associated with the given shared memory.

The size of the terminal must have at least 40 rows and 145 columns. It is decomposed in four sub-windows:

  • The "city map and status" window (upper-left)
  • The "character information" window (upper-right)
  • The "mailbox content" window (bottom-left)
  • The "enemy country monitor" window (bottom-right) 'Q', 'q' and 'Esc' keys are used to exit from the TUI.
