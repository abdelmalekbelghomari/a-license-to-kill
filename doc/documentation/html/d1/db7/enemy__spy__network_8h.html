Description détaillée

Header file for the Enemy Spy Network executable.

The Enemy Spy Network executable manages the spy and case officer threads in the espionage network. Their goal is to steal information and send it to the enemy country, place the information in a mailbox, and have the case officer retrieve the encrypted letters and send them to the enemy country.
