\doxysubsection{Description détaillée}
Creates a new \char`\"{}\+Terminal User Interface\char`\"{} using ncurses and associated with the given shared memory. 

The size of the terminal must have at least 40 rows and 145 columns. It is decomposed in four sub-\/windows\+:
\begin{DoxyItemize}
\item The \char`\"{}city map and status\char`\"{} window (upper-\/left)
\item The \char`\"{}character information\char`\"{} window (upper-\/right)
\item The \char`\"{}mailbox content\char`\"{} window (bottom-\/left)
\item The \char`\"{}enemy country monitor\char`\"{} window (bottom-\/right) \textquotesingle{}Q\textquotesingle{}, \textquotesingle{}q\textquotesingle{} and \textquotesingle{}Esc\textquotesingle{} keys are used to exit from the T\+UI. 
\end{DoxyItemize}
