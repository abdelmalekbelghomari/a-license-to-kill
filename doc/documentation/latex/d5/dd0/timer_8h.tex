\doxysubsection{Description détaillée}
Header file for the Timer executable. 

The Timer executable is responsible for providing time and round information to other processes. It uses {\ttfamily S\+I\+G\+A\+L\+RM} to tick the clock and, if the maximum number of rounds is reached, it terminates all other processes using {\ttfamily S\+I\+G\+I\+NT}.
